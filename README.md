# Base Builder Starter

Simple Node.js script to interact with the Base network using ethers.js.

## Features
- Connects to Base Sepolia RPC
- Fetches latest block number
- Demonstrates basic on-chain interaction

## Tech Stack
- Node.js
- ethers.js v6
- Base Sepolia

## How to Run
```bash
npm install
node index.js
